Original Implementation

The original version of this project was implemented as a Python-based dashboard using:

  • Python with Jupyter Dash for the web interface
  • MongoDB for data storage and retrieval
  • Plotly and Leaflet for data visualization

This version required running a Python server and MongoDB instance locally, making deployment more complex.

Setup Instructions

Prerequisites

  • Python 3.8 or higher
  • MongoDB installed and running locally
  • pip (Python package installer)

Installation Steps

  1. Clone or download the project files
  2. Install required Python packages: 
    pip install jupyter-dash dash dash-leaflet plotly pandas numpy matplotlib pymongo
  3. Import the data into MongoDB: 
    mongoimport --db AAC --collection animals --file aac_shelter_outcomes.csv --type csv --headerline
  4. Run the Python script: 
    python original.py

Source Files

  • Download original.py - The main Python script
  • Download aac_shelter_outcomes.csv - The dataset
